# HomeMarket - מנוע חיפוש נכסים

מנוע חיפוש נכסים מתקדם ומודרני עבור HomeMarket, עם אינטגרציה ישירה לגיליון Google Sheets, פילטרים מתקדמים ועיצוב יפהפה.

## תכונות

- **אינטגרציה עם Google Sheets**: התחברות ישירה לגיליון הנכסים שלכם
- **טעינה אוטומטית**: הכלי יטען נתונים ישירות מהגיליון הציבורי
- **פילטרים מתקדמים**: חיפוש לפי טקסט, עיר, שכונה, טווח מחירים, חדרים, סוג נכס, סטטוס, מגייס ופרויקט
- **עיצוב רספונסיבי**: עיצוב mobile-first עם פילטרים מתקפלים וכרטיסים ידידותיים למגע
- **נכסים מומלצים**: אדמין יכול לסמן נכסים חשובים בראש הרשימה
- **עדכונים בזמן אמת**: סינכרון נתונים חי עם Google Sheets
- **ממשק מודרני**: עיצוב נקי עם מיתוג HomeMarket ואנימציות חלקות

## התקנה והפעלה

1. התקן dependencies:
   ```bash
   npm install
   ```

2. הפעל שרת פיתוח:
   ```bash
   npm run dev
   ```

3. בנה לייצור:
   ```bash
   npm run build
   ```

## הגדרת Google Sheets API (אופציונלי)

הכלי יעבוד ישירות עם הגיליון הציבורי, אך לביצועים טובים יותר ניתן להגדיר מפתח API:

1. עבור ל-[Google Cloud Console](https://console.cloud.google.com/)
2. צור פרויקט חדש או בחר קיים
3. הפעל את Google Sheets API
4. צור אישורים (מפתח API)
5. הגבל את מפתח ה-API ל-Google Sheets API
6. צור קובץ `.env` והוסף:
   ```
   VITE_GOOGLE_SHEETS_API_KEY=your_api_key_here
   ```
